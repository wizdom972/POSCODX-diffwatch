## 👁 diffwatch

GitHub에 코드가 push되면 자동으로 영향도를 분석하고 담당자에게 알림을 보내는 AI 에이전트입니다.

---

**🔍 주요 기능**

| 기능 | 설명 |
|------|------|
| 자동 감지 | GitHub Webhook으로 push 이벤트 실시간 수신 |
| 영향도 분석 | git diff를 읽고 높음 / 중간 / 낮음 자동 판정 |
| 담당자 알림 | 변경 파일 경로 기반으로 담당자 매핑 후 알림 발송 |
| 이력 기억 | 분석 결과를 RAG에 저장, 자연어로 과거 변경 이력 조회 |
| 대시보드 | 커밋 분석 현황 및 알림 이력 웹 UI로 확인 |
| 내장 챗봇 | 대시보드 안에서 에이전트와 실시간 스트리밍 대화 |

---

**🛠 사용 기술**

- **에이전트** — LangGraph + DeepAgents
- **검색** — LangChain RAG (변경 이력 벡터 저장 및 자연어 검색)
- **서버** — FastAPI (Webhook 수신 + 대시보드 API)
- **트리거** — GitHub Webhook
- **UI** — 커스텀 대시보드 (HTML / CSS / JS)

---

**📐 에이전트 구성**

```
Level 1  시스템 프롬프트 — 코드 영향도 분석 전문가 역할 정의
Level 2A 커스텀 도구    — git / 알림 / 변경 이력 저장 (총 8개)
Level 2C RAG          — 분석 결과 벡터 저장 및 자연어 검색
Level 2D 스킬         — 분석 절차 · 영향도 기준 · 보고서 형식
```
